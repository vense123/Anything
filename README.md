### ingress
分成两个策略。
1. ingress
没什么好说的，就是ingress游戏要连哪个。

2. ingress 验证
现在ingress监测IP，不行的直接进不去游戏。直连的话，中国电信可以，中国联通不行,中国移动的朋友均已携号转网 所以不清楚 :) 可以设置个能用的梯子，这样子就可以通过验证了。(长按图标可选服务器)

`ingress` 使用无法通过验证的梯子，`ingress验证` 使用通过验证的梯子。简单讲就是只要通过验证，进游戏niantic就不管你了。
